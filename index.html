HTML

HTML adalah kependekan dari Hyper Text Markup Language,
yaitu sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web.
Si HTML inilah yang menyusun sebuah halaman web menjadi sebagaimana
yang kita lihat melalui browser (penjelajah internet).

Fitur HTML:

  1. Mudah dipelajari dan mudah digunakan.
  2. HTML adalah platform independen.
  3. Gambar, video dan audio dapat ditambahkan ke halaman web.
  4. Hypertext dapat ditambahkan ke teks.
  5. HTML adalah bahasa markup.

Keuntungan:

Kerugian:
  1. HTML digunakan untuk membuat website.
  2. support semua web browser.
  3. bisa digunakan dengan bahasa pemrograman lain seperti CSS, JavaScript dll.
  1. HTML hanya dapat membuat halaman web statis sehingga untuk halaman web dinamis bahasa lain harus digunakan
  2. Sejumlah besar kode harus ditulis untuk membuat halaman web sederhana.
  3. Fitur keamanannya tidak bagus.
