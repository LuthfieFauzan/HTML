Tutorial basic membuat web

1. HTML

Setiap kode HTML harus dilampirkan di antara tag HTML dasar. Dimulai dengan dan diakhiri dengan tag .

2. Head

Tag Head yang berisi semua header informasi dari halaman web atau dokumen seperti judul halaman dan informasi lain-lain.
Informasi ini terlampir dalam tag kepala yang terbuka dengan < head > dan diakhiri dengan < /head >.

3. Title

Kita dapat menamakan judul halaman web menggunakan tag < title >. Ini adalah informasi tajuk dan karenanya disebutkan dalam tag tajuk.
Tag dimulai dengan < title > dan diakhiri dengan < /title >

4. Body

Langkah selanjutnya adalah yang paling penting dari semua tag yang telah kita pelajari sejauh ini.
Tag tubuh berisi badan halaman yang akan terlihat oleh semua pengguna. Ini terbuka dengan < body > dan diakhiri dengan < /body >.
Setiap konten yang terlampir dalam tag ini akan ditampilkan pada halaman web baik itu tulisan atau gambar atau audio atau video atau bahkan tautan.

5. Heading

Tag ini membantu webdev memberikan judul pada konten halaman web. Tag ini terutama ditulis di dalam tag body.
HTML memberi webdev enam tag heading dari < h1 > ke < h6 >. Setiap tag menampilkan tajuk dengan gaya dan ukuran font yang berbeda.

6. Paragraph

Tag ini membantu menulis pernyataan paragraf di halaman web. Mereka mulai dengan tag < p > dan berakhir dengan < p >.
Di sini tag < br > digunakan untuk memecah baris dan bertindak sebagai return carriage. < br > adalah tag kosong.

7. Images

Tag Images digunakan untuk menyisipkan gambar ke halaman web. Sumber gambar yang akan dimasukkan dimasukkan ke dalam tag < img src = "source_of_image" >.
